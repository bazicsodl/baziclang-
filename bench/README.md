# Benchmarks

This folder contains small, focused performance tests. Run with:

```powershell
.\bin\bazic.exe run .\bench\<name>.bz
```

## Included
- `loop_arith`
- `string_concat`
- `string_builder`
- `parse_int_float`
- `json_validate`
- `crypto_sha256`
- `base64_roundtrip`
- `match_hot`
- `jwt_sign_verify`
