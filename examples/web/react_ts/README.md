# Bazic UI React + TypeScript Adapter

This folder shows how to render the Bazic UI tree using React + TypeScript.

## Build Adapter
```powershell
npm install
npm run build:ui
```

## Run
1. Build the Bazic WASM:
```powershell
.\tools\web\build.ps1
```
2. Serve `examples/web` and open `react_ts/index.html`.
