# Reference Apps

This project targets v1.0 readiness with at least three reference applications.

## 1) CLI App
- Path: `examples/apps/cli`
- Purpose: demonstrate flags, file IO, and stdlib usage

## 2) Service App
- Path: `examples/apps/service`
- Purpose: demonstrate HTTP server + JSON responses

## 3) Desktop App
- Path: `examples/apps/desktop`
- Purpose: demonstrate native binary + simple UI launch

## 4) Web/WASM Demo
- Path: `examples/web`
- Purpose: demonstrate browser runtime and JS interop

Reference UIs under `examples/web/apps`:
- `apps/dashboard`
- `apps/form`
- `apps/landing`
- `apps/admin`

## 5) JWT Auth API
- Path: `examples/apps/jwtstack`
- Purpose: demonstrate JWT issuance, verification, and CSRF token generation
