Types

Bazic is statically typed. Types help you catch mistakes early.

Built-in types

  • int ? whole numbers
  • float ? decimals
  • bool ? true or false
  • string ? text
  • void ? no value (only for function returns)
  • any ? for interop only

Structs

struct Point { x: int; y: int; }
let p = Point{ x: 3, y: 4 };

Enums

enum Status { Ok, Bad, Unknown }
let s = Status.Ok;

Interfaces

interface Named { fn name(): string; }
impl User: Named;

Generics

struct Box[T] { value: T; }
fn identity[T](x: T): T { return x; }

No nil

Bazic does not allow nil. Use Option or Result.
