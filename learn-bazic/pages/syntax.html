Syntax Map

This page shows how Bazic code is shaped. The rules below match the Bazic v1.0 spec.

Keywords you will see

fn let if else while return struct enum interface impl match import true false

Comments

// line comment
/* block comment */

Blocks

if condition {
    // block body
}

Declarations

import "std";

struct User { name: string; age: int; }

enum Status { Ok, Error }

interface Named { fn name(): string; }

impl User: Named;

fn greet(name: string): string { return "Hi " + name; }

let version = "1.0";

Expressions

let sum = 1 + 2 * 3;
let ok = a > b && c != d;

Match expression

let label = match status {
    Ok: "OK",
    Error: "ERROR",
};
