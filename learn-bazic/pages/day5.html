Day 5 ? Functions

Goal: put logic into reusable blocks.

Simple function

fn square(x: int): int {
    return x * x;
}

fn main(): void {
    println(str(square(5)));
}

Multiple parameters

fn greet(name: string, age: int): string {
    return "Hello " + name + " (" + str(age) + ")";
}
Try it: Write a function that checks if a number is even.
Checkpoint: You can define and call your own functions.
