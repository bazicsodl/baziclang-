Web Stack (Server Routing)

Bazic includes a convention-based HTTP server via http_serve_app.

Quickstart

.\bin\bazic.exe run .\examples\apps\webstack\main.bz

How routing works

Handlers are discovered by name:

fn get_root(req: ServerRequest): ServerResponse {
    return http_text(200, "hello");
}

Path parameters

fn get_posts_p_id(req: ServerRequest): ServerResponse {
    let id = http_params_get(req.params, "id");
    return http_text(200, "post " + id);
}

Reading query, headers, cookies

let page = http_query_get(req.query, "page");
let auth = http_header_get(req.headers, "Authorization");
let session = http_cookie_get(req.cookies, "session");

Return JSON

return http_json(200, "{"ok":true}");

Server limits

Use environment variables to tune limits:

BAZIC_HTTP_MAX_BODY
BAZIC_HTTP_READ_TIMEOUT_MS
BAZIC_HTTP_WRITE_TIMEOUT_MS
