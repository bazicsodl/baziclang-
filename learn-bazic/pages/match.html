Match

Match is Bazic?s safe switch. It is exhaustive for enums.

Match statement

enum State { On, Off }

match state {
    On: { println("on"); }
    Off: { println("off"); }
}

Match expression

let label = match state {
    On: "ON",
    Off: "OFF",
};

Guards

enum Temp { Cold, Warm, Hot }

let t = Temp.Warm;
let msg = match t {
    Cold if is_winter: "Cold winter",
    Cold: "Cold",
    Warm: "Warm",
    Hot: "Hot",
};
