Enums

Enums define a small set of choices.

Define an enum

enum Role { Admin, Member, Guest }

Use it

let r = Role.Member;

Handle with match

match r {
    Admin: { println("admin"); }
    Member: { println("member"); }
    Guest: { println("guest"); }
}
