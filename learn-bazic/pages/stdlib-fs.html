Stdlib ? fs

std/fs

Read, write, and manage files.

Read a file

let res = fs_read_file("notes.txt");
if res.is_ok { println(res.value); }

Write a file

let ok = fs_write_file("notes.txt", "hello");
if !ok.is_ok { println(ok.err.message); }

List a directory

let res = fs_list_dir(".");
if res.is_ok { println(res.value); }

Safety

Never write paths from untrusted input without validation.
