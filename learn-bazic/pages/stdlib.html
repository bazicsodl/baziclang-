Standard Library (std)

The stdlib ships as a local package named std.

import "std";

Core modules

io
Read from input.
fs
Read, write, and list files.
time
Time, sleep, RFC3339.
json
Validate, minify, pretty-print.
http
Client + server helpers.
crypto
Hashing and secure tokens.
db
SQLite and driver-based DB access.
collections
StringBuilder helpers.
auth/jwt/session
Auth and tokens.
sql
SQL helper functions.
validate
Input validation helpers.
os/path
Environment and paths.
web/ui
WASM helpers.

Read a file

import "std";

fn main(): void {
    let res = fs_read_file("README.md");
    if res.is_ok { println(res.value); }
}

HTTP request

import "std";

fn main(): void {
    let res = http_get("https://example.com");
    if res.is_ok { println(res.value); }
}

Database query

import "std";

fn main(): void {
    let ok = db_exec("app.db", "create table if not exists notes(id int, text string);");
    if !ok.is_ok { println(ok.err.message); return; }
    let rows = db_query("app.db", "select id, text from notes;");
    if rows.is_ok { println(rows.value); }
}

Stdlib safety notes

  • Validate input before using it in file paths or queries.
  • Prefer parameterized queries with db_*_params.
  • Use crypto_bcrypt_hash for passwords.

See std/README.md for the full API list.
