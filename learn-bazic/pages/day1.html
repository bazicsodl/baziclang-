Day 1 ? Printing and first programs

Goal: run a program that prints words. That is enough for day 1.

Your first file

fn main(): void {
    println("Hello, Bazic!");
}

What each part means

  • fn main(): void defines the entry point.
  • { } wraps the block of code.
  • println prints with a newline.
Try it: Change the text to your name and run again.

print vs println

fn main(): void {
    print("Hello");
    print(" ");
    println("world");
}
Checkpoint: You can run a Bazic program and see output.

Mini practice

  1. Print your favorite food.
  2. Print two lines using println.
  3. Print three words on the same line using print.
