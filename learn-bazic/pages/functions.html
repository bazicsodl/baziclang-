Functions

Functions group logic so you can reuse it.

Signature shape

fn name(param: Type): ReturnType { ... }

Example

fn add(a: int, b: int): int {
    return a + b;
}

Entrypoint

Every program must have:

fn main(): void { ... }

Method call sugar

Calling user.name() is sugar for User_name(user) if it exists.

fn User_label(u: User): string { return u.name; }
println(user.label());
