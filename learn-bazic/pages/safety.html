Safety Model

Bazic is designed to make mistakes loud and behavior predictable.

No nil

Bazic rejects nil. Use Option or Result instead.

Explicit errors

let res = parse_int("42");
if res.is_ok {
    println(str(res.value));
} else {
    println(res.err.message);
}

No implicit conversion

Numbers and strings do not auto-convert. Use str or parsing helpers.

Exhaustive match

When you match on an enum, you must handle every variant.

Lint rules

The linter discourages using any except at boundaries.
