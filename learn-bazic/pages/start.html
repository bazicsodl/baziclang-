Start Here

Bazic is built to be friendly to beginners. This page gets you running today, even if you have never coded before.

Promise: If you do one tiny lesson per day, you will be writing Bazic comfortably in a week.

Step 1: Install Bazic

From the Bazic folder, run the installer once.

cd C:\Users\Ipeh\Documents\baziclang
.\tools\install.ps1
.\bin\bazic.exe version

On macOS or Linux, use ./tools/install.sh.

Step 2: Make your first project

.\bin\bazic.exe new hello
cd hello

This creates a folder with main.bz.

Step 3: Run it

.\bin\bazic.exe run

What a Bazic program looks like

fn main(): void {
    println("Hello, Bazic!");
}

The program always starts in main. Every line ends with ;.

Build a real executable

.\bin\bazic.exe build -o .\bin\hello.exe

Learning rules that keep you winning

  • Write tiny programs. Small wins build real skill.
  • Run code after every small change.
  • Type code yourself. Copying does not build memory.
  • Ask ?what does this line do?? until every line makes sense.
