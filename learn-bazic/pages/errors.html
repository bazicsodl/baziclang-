Errors, Option, Result

Bazic does not allow nil. Instead you use safe types.

Option

fn find_name(): Option[string] {
    return some("Ada");
}

let name = find_name();
let value = unwrap_or(name, "unknown");
println(value);

Result

let res = parse_int("42");
if res.is_ok {
    println(str(res.value));
} else {
    println(res.err.message);
}

Error values

let bad = err(0, Error{ message: "boom" });
Why it matters: you always know what can fail, and you handle it directly.
