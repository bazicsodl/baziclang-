Projects to build

These projects are small but real. Each one teaches a new part of the language.

1) CLI journal

  • Append text to a file.
  • Read and print all entries.
  • Use time_now_rfc3339 for timestamps.

2) JSON formatter

  • Read a JSON file.
  • Validate it with json_validate.
  • Pretty-print with json_pretty.

3) Mini web server

  • Use http_serve_text for a hello route.
  • Serve JSON with http_serve_json.

4) Notes app with SQLite

  • Create a table with db_exec.
  • Insert a note and list all notes.

5) Tiny UI app (WASM)

  • Create a UI project with bazic ui init.
  • Build a counter with ui_button and state.
