Structs

Structs bundle fields together to model real data.

Define a struct

struct User {
    name: string;
    age: int;
}

Create a value

let u = User{ name: "Ada", age: 21 };
println(u.name);

Generic structs

struct Box[T] { value: T; }
let b = Box[int]{ value: 42 };
