Day 3 ? If and else

Goal: make choices in code.

Comparisons

let a = 7;
let b = 3;
println(str(a > b));

If / else

fn main(): void {
    let age = 16;
    if age >= 18 {
        println("adult");
    } else {
        println("minor");
    }
}

Boolean logic

let ok = true;
let logged_in = false;
if ok && !logged_in { println("Please sign in"); }
Try it: Ask for a score variable and print ?pass? for 60+.
Checkpoint: You can branch based on conditions.
