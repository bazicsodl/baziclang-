Interfaces

Interfaces describe behavior. A struct can claim it supports that behavior with impl.

Define an interface

interface Named {
    fn name(): string;
}

Implement it

struct User { name: string; }
impl User: Named;

Method call sugar

Calling user.name() is sugar for a function named User_name(user).

fn User_name(u: User): string { return u.name; }
let u = User{ name: "Ava" };
println(u.name());
