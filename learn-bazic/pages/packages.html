Packages

Bazic uses a local package manager for safe, reproducible builds.

Initialize a project

.\bin\bazic.exe pkg init myapp

Add a local dependency

.\bin\bazic.exe pkg add stdutil .\examples\packages\stdutil
.\bin\bazic.exe pkg sync

Verify integrity

.\bin\bazic.exe pkg verify
.\bin\bazic.exe pkg sbom -o .\bazic.sbom.json

Import by alias

import "stdutil";
Security: Bazic verifies checksums and lockfile signatures on every build.
