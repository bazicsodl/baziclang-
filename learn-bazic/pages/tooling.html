Tooling

Bazic includes a full toolchain for building, testing, and shipping.

Run and build

.\bin\bazic.exe run
.\bin\bazic.exe build -o .\bin\app.exe

Check and format

.\bin\bazic.exe check .
.\bin\bazic.exe fmt .
.\bin\bazic.exe fmt --check .

Testing and linting

.\bin\bazic.exe test .
.\bin\bazic.exe test --filter match .
.\bin\bazic.exe test --json .
.\bin\bazic.exe lint .

Package manager

.\bin\bazic.exe pkg init myapp
.\bin\bazic.exe pkg add stdutil .\examples\packages\stdutil
.\bin\bazic.exe pkg sync
.\bin\bazic.exe pkg verify

Backends

  • LLVM backend for native builds. Requires clang.
  • Go backend for WASM and cross-platform builds.

UI helpers

.\bin\bazic.exe ui init --dir .\my-ui
.\bin\bazic.exe ui dev --dir .\my-ui --port 8080

Helpful environment variables

  • BAZIC_BACKEND=go to default to Go backend.
  • BAZIC_CLANG to point to clang.
  • BAZIC_TARGET=wasm for WASM builds.
