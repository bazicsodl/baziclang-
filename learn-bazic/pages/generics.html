Generics

Generics let you write reusable code that works with many types.

Generic struct

struct Box[T] { value: T; }
let i = Box[int]{ value: 1 };
let s = Box[string]{ value: "hi" };

Generic function

fn identity[T](x: T): T { return x; }

Generic bounds

interface Named { fn name(): string; }
fn greet[T: Named](x: T): string { return "Hi " + x.name(); }
