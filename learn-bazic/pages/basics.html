Basics of Bazic

This is the short version of the language. Use it when you want a fast refresher.

Every program starts at main

fn main(): void {
    println("Start here");
}

Variables

let name: string = "Rin";
let age = 12;

Control flow

if age >= 18 { println("adult"); } else { println("minor"); }

Loops

let i = 0;
while i < 3 { println(str(i)); i = i + 1; }

Functions

fn add(a: int, b: int): int { return a + b; }

Structs and enums

struct User { name: string; }
enum Role { Admin, User }

Match

match Role.Admin { Admin: { println("admin"); } User: { println("user"); } }
