Day 7 ? Build a mini app

Goal: build a tiny note app that reads and writes a file.

What you will build

  • Read note.txt.
  • Append a new line.
  • Save it back.

Mini app code

import "std";

fn main(): void {
    let res = fs_read_file("note.txt");
    let current = "";
    if res.is_ok { current = res.value; }

    let line = "New note at " + time_now_rfc3339();
    let next = current + "
" + line;

    let saved = fs_write_file("note.txt", next);
    if saved.is_ok {
        println("Saved note");
    } else {
        println(saved.err.message);
    }
}
Try it: Change the line to include your name.
Checkpoint: You can build a small app using the stdlib.
