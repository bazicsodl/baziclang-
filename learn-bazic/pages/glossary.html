Glossary

Simple definitions for the words you will see in Bazic.

Term list

  • Compiler ? A tool that turns code into a program you can run.
  • Binary ? A finished program file (like .exe).
  • Type ? The kind of data, like int or string.
  • Struct ? A custom data type with named fields.
  • Enum ? A small, fixed set of options.
  • Function ? A named block of logic you can call.
  • Result ? A value that is either ok or an error.
  • Option ? A value that is either some or none.
  • Match ? A safe switch for enums.
  • Backend ? The output engine that builds the program.
