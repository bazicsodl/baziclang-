Full Learning Track (Zero to Real Apps)

This track is built for complete beginners. Each day is a short, focused lesson with practice.

Week 1 ? Core language

  1. Day 1: Print, run, and read output.
  2. Day 2: Numbers, strings, and variables.
  3. Day 3: If, else, and boolean logic.
  4. Day 4: While loops and counters.
  5. Day 5: Functions and return values.
  6. Day 6: Structs for modeling data.
  7. Day 7: Enums and match safety.

Week 2 ? Real programs

  1. Day 8: Errors, Option, Result, and safe flow.
  2. Day 9: Filesystem tools with std/fs.
  3. Day 10: JSON and HTTP basics.
  4. Day 11: Build a CLI tool.
  5. Day 12: Build a web server using http_serve_app.
  6. Day 13: SQLite basics with std/db.
  7. Day 14: Build a small project end to end.

Week 3 ? Advanced but practical

  1. Day 15: Interfaces and method-call sugar.
  2. Day 16: Generics and constraints.
  3. Day 17: Packages, lockfiles, and verification.
  4. Day 18: Tooling, tests, and linting.
  5. Day 19: Web + UI build pipeline.
  6. Day 20: Performance and safety checklist.
