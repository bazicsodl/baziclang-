Day 2 ? Numbers and variables

Goal: store values in variables and do basic math.

Numbers

fn main(): void {
    let a: int = 10;
    let b: int = 3;
    let c = a + b;
    println(str(c));
}

Types you will use today

  • int for whole numbers.
  • float for decimals.
  • string for text.

Inferred types

let name = "Sam";   // inferred as string
let score = 98;     // inferred as int
let price = 3.14;   // inferred as float
Try it: Write a program that calculates the total price of 3 items.

Concatenating strings

fn main(): void {
    let who = "Ava";
    println("Hello, " + who);
}
Checkpoint: You can create variables and use math and strings.
