Control Flow

Control flow is how you choose or repeat work.

If / else

if score >= 60 {
    println("pass");
} else {
    println("try again");
}

While loops

let i = 0;
while i < 3 {
    println(str(i));
    i = i + 1;
}

Match statement

enum Mode { Dev, Prod }

fn main(): void {
    let m = Mode.Dev;
    match m {
        Dev: { println("dev"); }
        Prod: { println("prod"); }
    }
}

Match expression

let label = match m {
    Dev: "Dev mode",
    Prod: "Prod mode",
};
