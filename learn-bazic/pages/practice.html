Practice drills

Short exercises that build real skill. Pick three per day.

Basics

  1. Print your name 5 times using a loop.
  2. Create variables for height, age, and favorite color.
  3. Convert a number to a string using str.

Functions

  1. Write is_even that returns bool.
  2. Write max that returns the bigger of two ints.
  3. Write greet that returns a greeting string.

Structs and enums

  1. Make a Book struct with title and pages.
  2. Create an enum for Size with Small, Medium, Large.
  3. Use match to print a message for each size.

Stdlib

  1. Read a file and print it.
  2. Write a file with today?s date using time_now_rfc3339.
  3. Validate JSON with json_validate.
