Day 4 ? Loops

Goal: repeat work using while.

Counting loop

fn main(): void {
    let i = 1;
    while i <= 5 {
        println(str(i));
        i = i + 1;
    }
}

Loop with a total

fn main(): void {
    let i = 1;
    let total = 0;
    while i <= 5 {
        total = total + i;
        i = i + 1;
    }
    println(str(total));
}
Try it: Print the first 10 even numbers.
Checkpoint: You can repeat code and update variables.
