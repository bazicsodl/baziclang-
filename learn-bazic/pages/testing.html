Testing

Bazic includes a simple test runner built into the CLI.

Test file and naming

// file: math_test.bz
fn test_add(): bool {
    return add(2, 3) == 5;
}

Run tests

.\bin\bazic.exe test .
.\bin\bazic.exe test --filter add .
.\bin\bazic.exe test --json .

Assertions

fn test_parse(): void {
    let r = parse_int("10");
    assert(r.is_ok);
    assert_msg(r.value == 10, "expected 10");
}
