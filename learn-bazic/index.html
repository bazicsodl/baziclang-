How Bazic feels

Simple syntax

Few keywords. Clear punctuation. Easy to read at a glance.

Static types

Catch mistakes early with explicit types and local inference.

No hidden nulls

Use Option and Result instead of nil.

Fast builds

AOT compilation to native binaries or WASM.

Quick install

Windows

cd C:\Users\Ipeh\Documents\baziclang
.\tools\install.ps1
.\bin\bazic.exe version

macOS / Linux

cd ~/baziclang
./tools/install.sh
./bin/bazic version

If you already have the compiler, skip install and jump straight to the lessons.

Language snapshot

Structs + functions

struct User { name: string; age: int; }

fn greet(u: User): string {
    return "Hello " + u.name;
}

Enums + match

enum Role { Admin, Member, Guest }

fn badge(r: Role): string {
    return match r {
        Admin: "ADMIN",
        Member: "MEMBER",
        Guest: "GUEST",
    };
}
